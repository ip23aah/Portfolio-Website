# Portfolio Website

A professional portfolio website showcasing my projects, technical skills, and academic achievements. Built with vanilla HTML, CSS, and JavaScript following accessibility and usability best practices.

## 📋 Project Overview

This portfolio website was developed as part of my Computer Science degree coursework, demonstrating proficiency in web development fundamentals, accessibility standards, and user experience design.

### Key Features

- *Responsive Design*: Fully responsive across desktop, tablet, and mobile devices
- *Accessibility Compliant*: WCAG 2.1 Level AA compliant with 0 accessibility errors
- *Interactive Elements*: Modal popups, smooth animations, and dynamic content loading
- *Professional Presentation*: Clean, modern design suitable for job applications
- *Performance Optimized*: Fast loading times with optimized assets

## 🛠️ Technical Stack

- *HTML5*: Semantic markup and modern HTML features
- *CSS3*: Custom properties, Flexbox, Grid, animations
- *JavaScript (ES6+)*: Vanilla JavaScript with modern features
- *Accessibility*: ARIA attributes, keyboard navigation, screen reader support
- *Version Control*: Git with regular commits and meaningful messages

## 🔧 Installation & Setup

1. *Clone the repository*
   bash
   git clone https://github.com/ip23aah/Portfolio-Website.git
   cd portfolio-site
   

2. *Open in your preferred editor*
   bash
   code . # VS Code
   

3. *Run locally*
   - Open index.html in your browser, or
   - Use Live Server extension in VS Code, or
   - Run a local server: python -m http.server 8000

4. *View the website*
   - Navigate to http://localhost:8000 (if using Python server)
   - Or simply open index.html in your browser
